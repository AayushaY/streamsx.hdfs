Composites

composite WebHdfsRead(output OutStream)

WebHdfsRead -- Read a file from HDFS via the REST APIs

All operators in this composite will be fused into a single PE.

Parameters

  • url: The base url, should end with http://.../webhdfs/v1
  • username
  • password
  • file: the file to be read
  • format: format to use for the parse operator. Defaults to Parse.line
  • blockSize: size of block to get per http call. (optional)
  • OutType: the type of the output stream.
  • partition: (optional) partition colocation id to use. Use this if you're doing higher-level fusion in your application.

Output Ports

  • OutStream

composite WebHdfsReadFiles(output OutStream; input FilenameStream)

WebHdfsReadFiles -- Read files given on an input stream from HDFS via the REST APIs

All operators in this composite will be fused into the same PE.

Parameters

  • url: The base url, should end with http://.../webhdfs/v1
  • username: username to use to connect, if empty, it's ignored
  • password: password to use to connect, optional.
  • format: format to use for parse operator. Defaults to Parse.line
  • blockSize: size of block to get per http call.
  • OutType: the type of output stream.
  • partition: (optional) the name of the partition to use in the partitionColocation statement

Input Ports

  • FilenameStream

Output Ports

  • OutStream
