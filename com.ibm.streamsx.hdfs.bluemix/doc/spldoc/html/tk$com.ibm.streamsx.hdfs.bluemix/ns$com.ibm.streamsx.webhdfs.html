Namespace com.ibm.streamsx.webhdfs

Toolkits > com.ibm.streamsx.hdfs.bluemix 0.3.1 > com.ibm.streamsx.webhdfs

Operators

  • WebHdfsDirectoryScan: Scan a HDFS directory using WebHdfs.
  • WebHdfsRead: WebHdfsRead -- Read a file from HDFS via the REST APIs All operators in this composite will be fused into a single PE.
  • WebHdfsReadBinaryFiles: Read binary data from a webhdfs file.
  • WebHdfsReadFiles: WebHdfsReadFiles -- Read files given on an input stream from HDFS via the REST APIs All operators in this composite will be fused into the same PE.
  • WebHdfsWrite: Write data to WebHdfs Formats the tuple with Format, and then sends the blob to WebHdfsWriteBinary.
  • WebHdfsWriteBinary: Used by WebHdfsWrite; the only case you should consider using this is if your data is binary blobs so that you don't want a format operator.
