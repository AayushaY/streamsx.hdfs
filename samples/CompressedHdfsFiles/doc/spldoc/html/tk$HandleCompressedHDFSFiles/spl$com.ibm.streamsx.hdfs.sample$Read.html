composite Read

Read compressed data from HDFS with the Decompress and Parse operators.

Before running this composite, run the Write composite to write data into HDFS.

Once the file <code>testCompress.gz</code> exists, this application will read the file and output the results in csv format.

The steps are:
  • Read the binary data from HDFS
  • Decompress the data blocks.
  • Parse the data blocks into tuples

We then write the tuples to disk so that you can verify they are what we compressed in the first place.
